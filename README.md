## -[hsv](bgr2hsv.py)    
     
hsv를 사용하는 이유는 bgr은 색을 특정하기 어렵지만 hsv는 특정색을 찾기 쉽기 때문이다.      
그중 h는 색을 나타내는데 360이 아니라 0~179인 이유는 opencv가 255이상을 표현하지 못하기 때문에 절반으로 나눈것이다.     
(sv는 명도와 채도를 나타낸다.)
